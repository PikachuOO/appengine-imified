Using IMIfied and Google App Engine to build XMPP Applications

Providing an Instant Messaging interface to your application is easier than you might think. And you don't have to wait for the upcoming XMPP API or ignore those using other networks.

IMIfied provide a currently free service which provides a web hook like interface to an instant messaging bot. Try sending a message to appengine-imified@bot.im. This is just an example, you can find the code to add to your own applications at githib.com/garethr/appengine-imified/

{% if messages %}
    {% for message in messages %}

  • {{message.userkey}} ({{message.network}})

    {{message.msg}}

    • {% endfor %}
{% endif %} {% if debug %} {% include "_form.html" %} {% endif %}

This application was originally build by Gareth Rushgroge of morethanseven.net. The code can be found on GitHub.
